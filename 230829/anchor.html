두번째 문단

2. 다운이 완료되면 vs code로 들어가기 (노드js 다운후 설치과정을 미쳐 스크린샷하지 못했다..) Node js를 설치하면 자동으로 공식 패키지 매니저인 NPM(Node.js Package Manager)가 포함되어있는데 NPM은 라이브러리, 모듈, 프레임워크 등을 제공하고 쉽게 설치, 관리할 수 있도록 도와준다. NPM을 통해서 먼저 package.json을 생성 해야하는데 package.json 파일은 Node.js 프로젝트에서 필요한 패키지 정보와 프로젝트 설정 정보를 담고 있는 파일이다. [출처] 심플하게 익히자 vs code에 Node.js 환경 연결하기|작성자 Yoon Hyejin

세번째 문단

디버깅 실행을 하는 이유는 프로그램이 정상적으로 작동하지 않을 때 코드 상의 문제점을 파악하고 수정하기 위해 사용한다. 작업을 매번 완벽하게 할 수 없고 오류를 예측하고 찾아낼 수 없기때문에 디버깅 실행을 통하여 코드 작성시 오류를 최소화하고 안정성과 효율성을 높여 완성도 높은 작업물을 만들 수 있다.
